# Шашки с анимацией

## Описание проекта

Вам предоставлен шаблон для игры в шашки, включающий:
- HTML-разметку с игровым полем
- Базовые CSS-стили для доски и шашек
- JavaScript-заготовку с функционалом создания доски и начальной расстановки фигур


## Задача

Реализовать игру в шашки с анимацией следующих элементов:

### 1. Ход шашек

**Требования:**
- Реализовать логику перемещения шашек по доске
- При перемещении шашка должна плавно анимироваться к новой позиции
- Использовать **Event Delegation** для обработки кликов на шашках и ячейках доски

**Технология анимации:** CSS `transition`

### 2. Убийство шашек

**Требования:**
- Реализовать логику "съедания" шашек противника при перепрыгивании через них
- При удалении шашка должна анимироваться, например, исчезать с эффектом масштабирования или затухания (Можно использовать гифку из `media/`)
- Удалять шашку из DOM после завершения анимации

**Технология анимации:** CSS `@keyframes`

### 3. Уведомление о победе

**Требования:**
- Определять момент победы одного из игроков (когда у противника не осталось шашек)
- Создавать элемент уведомления о победе **динамически через JavaScript** (в исходной верстке таких элементов нет)
- Отображать информацию о победителе

**Технология:** JavaScript DOM API


## Критерии оценки

1. Корректная работа логики ходов шашек
2. Плавная анимация перемещения (CSS transition)
3. Корректная логика "съедания" шашек
4. Анимация удаления шашек (CSS keyframes)
5. Определение победителя
6. Создание уведомления о победе (JavaScript DOM)
7. Чистота и читаемость кода
